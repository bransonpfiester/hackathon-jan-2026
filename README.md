# Hackathon Projects (Jan 30 - Feb 1, 2026)

A collection of projects built during the January 2026 hackathon.

## Projects

### 01 Pomodoro Timer
A simple browser-based Pomodoro timer with:
- 25-minute focus sessions / 5-minute breaks
- Task input before each focus session
- Adjustable timer duration
- 9 customizable themes
- Sound notifications

**Run it:** Open `01 Pomodoro Timer/index.html` in your browser.
